# SEO 2026 Research for Casino WordPress Themes

## Key SEO 2026 Requirements

### 1. Schema.org Structured Data
- GamblingService schema
- FAQ schema
- Review/Rating schema
- Event schema (for live games)
- BreadcrumbList schema
- Organization schema with license info
- WebSite schema with SearchAction

### 2. Core Web Vitals (CWV)
- LCP (Largest Contentful Paint) < 2.5s
- INP (Interaction to Next Paint) < 200ms
- CLS (Cumulative Layout Shift) < 0.1
- Lazy loading images
- Preload critical CSS/fonts
- Minified assets

### 3. Mobile-First Architecture
- Responsive design (320px to 4K)
- Touch-friendly UI (min 44px tap targets)
- Bottom navigation for mobile
- PWA-ready (manifest.json, service worker)
- AMP-compatible pages

### 4. E-E-A-T Signals
- Author bio sections
- License/regulation badges in header/footer
- Trust indicators (SSL, responsible gambling)
- Expert review sections
- About page with credentials

### 5. AI/LLM Optimization
- Semantic HTML5 structure
- Answer Engine Optimization (AEO) - 50-100 word summary blocks
- Topic clusters instead of single keywords
- Machine-readable content structure

### 6. Technical SEO
- Canonical URLs
- Hreflang for multi-language
- XML sitemap auto-generation
- Robots.txt optimized
- Open Graph + Twitter Cards
- Preconnect to CDN/API domains
- Critical CSS inline

### 7. Content SEO
- H1-H6 hierarchy
- Internal linking structure
- Breadcrumbs
- Table of contents for long content
- Image alt text with keywords
- Video schema for embedded content

### 8. Performance
- Image optimization (WebP/AVIF)
- CSS/JS minification
- Font display: swap
- Above-the-fold content priority
- Defer non-critical JS
