### ข้อกำหนดหมวด SEO (7 ข้อ)
1. ระบบต้องเป็น **SEO Specialist Agentic AI** ที่เข้าใจทุกอย่างทำงาน SEO ได้แค่ user add domain → ระบบทำ SEO ให้ตั้งแต่เริ่มแบบครบวงจร SEO สายเทา แบบ specialist
2. หมวดนี้ user จะไม่เห็นโดเมนของกันและกันยกเว้น **super admin** ที่เห็นหมด
3. user จะเห็น PBN ทั้งหมดเหมือนกันแต่มีเฉพาะแค่ **super admin** เท่านั้นที่เพิ่มหรือลบได้
4. เป้าหมายปลายทางคือทำ SEO ให้ **ติดหน้าแรกภายในระยะเวลาอันสั้นไม่เกิน 3-7 วัน** สามารถใช้ได้ทุกทางไม่จำกัดเทคนิคและต้องทำจริง
5. โดเมนในระบบอาจมีถึง **150-200 โดเมน** เพราะฉะนั้นจัดการงานให้ดีไม่ให้เนื่องจาก user ใช้งานหลายคน
6. หลังจาก user เพิ่ม domain แล้ว **AI สามารถเลือกใช้ PBN** ในการทำ อันดับ SEO ได้และไปหาจากเว็บ น่าเชื่อถือสูงแล้วโพสได้เหมือนกัน
7. ส่วนนี้ให้เพิ่มว่าต้องการ **ติดอันดับเร็วแค่ไหน 3วัน 7 วัน หรือ 30 วัน** ตัดส่วน อุตสาหกรรมออก ระยะเวลาที่ตีดให้ AI ประเมินจาก **keywords ความยาก ง่าย** ต้องความก้าวร้าวให้ เปลี่ยนเป็น

### UI "เพิ่มโดเมนใหม่" (หน้า 2 — dialog mockup)
- **โดเมน**: text input (example.com)
- **Niche / อุตสาหกรรม**: text input (e.g. crypto, health, finance)
- **Keywords เป้าหมาย**: text input คั่นด้วย comma — AI จะใช้ keywords เหล่านี้ในการทำ SEO
- **Strategy**: dropdown (Grey Hat — สมดุลระหว่างความเสี่ยงและผลลัพธ์)
- **ความก้าวร้าว: 5/10**: slider (Conservative → Moderate → Aggressive)
- **WordPress Connection (ไม่บังคับ)**: ใส่ Application Password เพื่อให้ AI แก้ไขเว็บไซต์จริง
  - WordPress Username
  - Application Password (xxxx xxxx xxxx xxxx xxxx xxxx)
  - สร้างได้ที่ WordPress Admin → Users → Profile → Application Passwords
- **Auto Backlink Building**: toggle — AI สร้าง backlink อัตโนมัติ
- **Auto Content Creation**: toggle — AI สร้างเนื้อหา SEO อัตโนมัติ
- **Auto PBN Posting**: toggle — โพสต์ไปยัง PBN network อัตโนมัติ
